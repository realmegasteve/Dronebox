<div align="center">
Dronebox is a mod that adds drones for the modding event Modfest: Toybox <br> <br>
This mod has multiple accessories to allow players to give their drones amazing new features to allow for revamping the player's gameplay like: <br> <br>
  
Lamps, fireworks as boosters, googly eyes and even more amazing accessories you can find while playing the mod
</div>


## <div align="center">features</div>
<div align="center">
this mod features<br>
a controller: for allowing more precise controll of the drone<br> 
a remote: to allow you to target a specific location and the drone will then move towards it<br>
a python drive: allowing you to controll the drone with python<br>
a drone: the drone itself has to be controlled but is in itself an amazing device<br>
</div>
